# Cloudflare Decision

- Users automatically connect to the worker closest to them
- When you create a D1 database, by default, it is created at the data center closest to you
- Read replications can be created for the DB to improve performance
- Cloudflare CDN will serve the static frontend globally
- Cloudflare has multiple data centers worldwide
- Cloudflare has bot prevention & traffic blocking to protect our website
- Cloudflare has metrics to show unique visitors
  - Important for measuring the conversion rate
- Enable logging in Cloudflare
- Set a custom domain
